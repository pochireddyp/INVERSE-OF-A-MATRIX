### Step 2: Define the Matrix

### Step 4: Print the result to see the inverse matrix:

## Program:
```

#Developed by:pochireddy.p

import numpy as np
a=[[6,2,3],[3,1,1],[10,3,4]]
sol=np.linalg.inv(a)
print(sol)
```
